# INAPA Timeline Presentation

Presentacion web en React + Vite para exponer el cronograma operativo del proyecto SBCC-003-2024/INAPA-AFD-UE.

## Requisitos

- Node.js 18+
- npm

## Ejecutar en local

```bash
npm install
npm run dev
```

## Build de produccion

```bash
npm run build
npm run preview
```

## Navegacion durante la presentacion

- `←` / `→`: diapositiva anterior o siguiente
- `PageUp` / `PageDown`: navegar diapositivas
- `Space`: siguiente diapositiva
- `F`: activar/desactivar pantalla completa

## Estructura principal

- `src/App.jsx`: contenido y estructura de slides
- `src/App.css`: estilos y layouts de presentacion
- `src/index.css`: estilos globales base
